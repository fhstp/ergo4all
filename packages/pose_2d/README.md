# Ergo4All pose 2d package

This flutter package contains data types and logic for interacting with 2d pose
data.

## Installation

Reference this package in another part of the project by adding `pose_2d: any`
to the dependencies.

## Usage

This package allows you to project a previously normalized pose onto the
anatomical planes, resulting in 2d poses.

Use the functions in [the main file](lib/pose_2d.dart) for this.

## Contributing

In order to generate better release notes, please include the scope `pose_2d` in
your commit messages, eg. `fix(pose_2d): some problem`.

This package uses [very_good_analysis](https://pub.dev/packages/very_good_analysis)
for style rules. Please make sure all warnings are fixed before committing.
