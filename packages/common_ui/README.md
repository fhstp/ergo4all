# Ergo4All common ui package

This flutter package contains common ui code for all project apps.

## Installation

Reference this package in another part of the project by adding `common_ui: any`
to the dependencies.

## Package structure

This package is split into two parts:

- `theme`: contains common style and theme code
- `widgets`: contains reusable or pre-styled widgets.

## Contributing

In order to generate better release notes, please include the scope `common_ui` in
your commit messages, eg. `feat(common_ui): add nice widget`.

This package uses [very_good_analysis](https://pub.dev/packages/very_good_analysis)
for style rules. Please make sure all warnings are fixed before committing.
