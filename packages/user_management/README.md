# Ergo4All user management package

This flutter package contains functions for managing users.

## Installation

Reference this package in another part of the project by adding
`user_management: any` to the dependencies.

## Usage

Take a look around the exported modules. You will find functions for creating,
updating and querying users.

## Contributing

In order to generate better release notes, please include the scope
`user_management` in your commit messages, eg.
`fix(user_management): some problem`.

This package uses [very_good_analysis](https://pub.dev/packages/very_good_analysis)
for style rules. Please make sure all warnings are fixed before committing.
