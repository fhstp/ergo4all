# Ergo4All pose visualization package

This flutter package contains logic for visualizing poses.

## Installation

Reference this package in another part of the project by adding `pose_vis: any`
to the dependencies.

## Usage

Use the `Pose2dPainter` and `Pose3dPainter` `CustomPainter`s to paint poses.

## Contributing

In order to generate better release notes, please include the scope `pose_vis` in
your commit messages, eg. `fix(pose_vis): some problem`.

This package uses [very_good_analysis](https://pub.dev/packages/very_good_analysis)
for style rules. Please make sure all warnings are fixed before committing.
