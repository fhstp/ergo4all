# Ergo4All common package

This package contains common dart utility logic to be used throughout the project.

## Installation

Reference this package in another part of the project by adding `common: any`
to the dependencies.

## Contributing

In order to generate better release notes, please include the scope `common` in
your commit messages, eg. `feat(common): add a cool function`.

This package uses [very_good_analysis](https://pub.dev/packages/very_good_analysis)
for style rules. Please make sure all warnings are fixed before committing.
