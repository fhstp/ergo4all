# Ergo4all pose tester app

Utility app for testing pose detection, angle calculation and scoring.

## Usage

Load an image from the device gallery using the "Select image" button. Once the image is loaded you have access to the following features:

### View pose on image

The loaded image will be displayed, overlayed with the detected pose.
Tap the image to hide/show the pose.

### Share pose data

Use the app-bar menu to share pose data.

### View angles

Use the angle tab to view calculated angles for the loaded pose.

### View scores

Use the scores tab to view calculated scores for the loaded pose.
Scores a grouped hierarchically and colored to indicate how
good/bad the score is.

### View 2d poses

Use the sagittal, coronal and transverse pose to view
2d projections of the pose.

## Development

### Building
